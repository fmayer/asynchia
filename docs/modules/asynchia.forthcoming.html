asynchia.forthcoming¶

Facilities to refer to data that is not yet available.

Example:

# Usually acquired by a call that results in network I/O. # Global variable for demostration purposes. a = DataNotifier() def bar():

# Request result of network I/O. blub = yield a yield Coroutine.return_(blub)
def foo():
# Wait for completion of new coroutine which - in turn - waits # for I/O. blub = yield Coroutine.call_itr(bar()) print “yay %s” % blub

c = Coroutine(foo()) c.call() # Network I/O complete. a.submit(‘blub’)

class asynchia.forthcoming.Coroutine(itr, pcontext=None, datanotifier=None)¶

Create coroutine from given iterator. Yielding None will pause the co- routine until continuation by the given PauseContext. Yielding a DataNotifier will send the requested value to the coroutine once available. Yielding an instance of Coroutine.return_ will end execution of the coroutine and send the return value to any coroutines that may be waiting for it or calls any callbacks associated with it.

call()¶

Start (or resume) execution of the coroutine.

classmethod call_itr(itr)¶

Create a coroutine from the given iterator, start it and return the DataNotifier.

handle_result(result)¶

Internal.

class return_(obj=None)¶

Yield an instance of this to signal that the coroutine finished with the given return value (defaults to None).

Coroutine.send(data)¶

Send requested data to coroutine.

class asynchia.forthcoming.DataNotifier(socket_map)¶

Call registered callbacks and send data to registered coroutines at submission of data.

add_callback(callback)¶

Add callback (function that only receives the data upon submission as an argument).

add_coroutine(coroutine)¶

Add coroutine that waits for the submission of this data.

add_databack(callback)¶

Add databack (function that receives the the data-notifier data upon submission as arguments).

inject(data)¶

Submit data and ensure their callbacks are called in the main thread.

poll()¶

Poll whether result has already been submitted.

submit(data)¶

Submit data; send it to any coroutines that may be registered and call any data- and callbacks that may be registered.

classmethod threaded_coroutine(socket_map, fun, *args, **kwargs)¶

Run fun(*args, **kwargs) in a thread and return a DataNotifier notifying upon availability of the return value of the function.

wait(timeout=None)¶

Block execution of current thread until the data is available. Return requested data.

class asynchia.forthcoming.PauseContext¶

Collection of Coroutines which are currently paused but not waiting for any data. They are paused to prevent too much time to be spent in them, preventing possibly important I/O from being done.

pause(coroutine)¶

Add coroutine to the list of paused coroutines.

unpause()¶

Continue all paused coroutines.
