asynchia.util¶

Auxiliary functions.

class asynchia.util.GradualAverage(values=None)¶

Memory-efficient average to which values may gradually be added over time. Its value can be accessed via the avg member.

add_value(value)¶

Add value to the average.

add_values(values)¶

Add values to the average. This is more effective than using add_value multiple times.

class asynchia.util.IDPool¶

Pool that returns unique identifiers.

Identifierers obtained using the get method are guaranteed to not be returned by it again until they are released using the release method.

>>> pool = IDPool()
>>> pool.get()
0
>>> pool.get()
1
>>> pool.get()
2
>>> pool.release(1)
>>> pool.get()
1
>>>
get()¶

Return a new integer that is unique in this pool until it is released.

release(id_)¶

Release the id. It can now be returned by get again.

Will reset the IDPool if the last id in use is released.

reset()¶

Reset the state of the IDPool. This should only be called when no identifier is in use.

class asynchia.util.LimitedAverage(samples, values=None)¶

Average considering the last samples values added to it. Its value can be accessed via the avg member. Stores up to samples objects in a deque and thus has a higher memory usage than GradualAverage (and a different use case).

add_value(value)¶

Add value to the average.

add_values(values)¶

Add values to the average. This is more effective than using add_value multiple times.

asynchia.util.goodsize(maxsize)¶

Return biggest power of two that is less or equal to maxsize.

Powers of two are considered good values to be passed to the recv method of sockets. To quote the python documentation of the socket module:

For best match with hardware and network realities, the value of bufsize should be a relatively small power of 2, for example, 4096.
asynchia.util.is_closed(sock)¶

Find out whether a socked is closed or not. Does only work on non-blocking sockets!

asynchia.util.is_unconnected(sock)¶

Find out whether a socket has not yet been connected. Does only work on non-blocking sockets!

asynchia.util.parse_ip(string, default_port=-1)¶

Return (host, port) from input string. This tries to automatically determine whether the address is IPv4 or IPv6.

If no port is found default_port, which defaults to -1 is used.

>>> parse_ip('127.0.0.1:1234')
('127.0.0.1', 1234)
>>> parse_ip('[2001:0db8:85a3:08d3:1319:8a2e:0370:7344]:443')
('2001:0db8:85a3:08d3:1319:8a2e:0370:7344', 443)
asynchia.util.parse_ipv4(string, default_port=-1)¶

Return (host, port) from IPv4 IP.

asynchia.util.parse_ipv6(string, default_port=-1)¶

Return (host, port) from IPv6 IP.

asynchia.util.socketpair()¶

Return pair of connected sockets. Unlike socket.socketpair this is platform independant. However, if socket.socketpair is available, it is used here as well.
