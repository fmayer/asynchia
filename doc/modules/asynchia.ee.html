class asynchia.ee.Collector(onclose=None)¶

This is the base-class for all collectors. Collectors read up to nbytes bytes of data from the protocol passed to them.

add_data(prot, nbytes)¶

Override to read at most nbytes from prot and store them in the collector.

close()¶

Called after the last data has been added.

IMPORTANT NOTE: This may be called without the Collector being initalised, so you need to explicitely check whether it is when you want to clean up resources allocated in init.

init()¶

Called before the first data is added to the collector.

class asynchia.ee.CollectorQueue(collectors=None, onclose=None)¶

Write data to the first collector until CollectorFull is raised, afterwards repeat with next. When the CollectorQueue gets empty it raises CollectorFull.

add_collector(coll)¶

Add coll to queue.

add_data(prot, nbytes)¶

Add data to first collector until it is full.

close()¶

Close collectors contained in the queue.

finish_collector(coll)¶

Called when a collector raises CollectorFull.

full()¶

Called when the last collector in the queue raises CollectorFull.

Return True to prevent CollectorQueue from raising CollectorFull.

class asynchia.ee.FileInput(fd, length=None, buffer_size=9096, closing=True, onclose=None)¶

Input that buffers at most buffer_size bytes read from the passed fd, and sends them in a buffered way. This can be used to “directly” send data contained in a file.

close()¶

If FileInput is closing, close the fd.

classmethod from_filename(filename, mode='rb', *args, **kwargs)¶

Same as cls(fd, size, *args, **kwargs) while fd and size get constructed from the filename. If the mode attribute does not equal ‘rb’, the size cannot be determined efficienty.

tick(sock)¶

Send as much of the file as possible.

class asynchia.ee.Input(onclose=None)¶

Base-class for all Inputs. It implements __add__ to return an InputQueue consisting of the two operants.

close()¶

Called after the last tick.

IMPORTANT NOTE: This may be called without the Input being initalised, so you need to explicitely check whether it is when you want to clean up resources allocated in init.

init()¶

Called before the first tick.

tick(sock)¶

Abstract method to be overridden. This should send the data contained in the Input over sock.

class asynchia.ee.InputQueue(inputs=None, onclose=None)¶

An InputQueue calls the tick method an object until it raises InputEOF, then it goes on with the next object in the queue. If no object is in the queue anymore, InputQueue.tick raises InputEOF (like any other Input that doesn’t need data anymore).

add(other)¶

Add input to queue.

close()¶

Close all inputs contained in the queue.

eof()¶

Called when the last Input in the queue raises InputEOF.

Return True to prevent InputQueue from raising InputEOF.

tick(sock)¶

Call tick method of the first object contained in the queue until it raises InputEOF.

class asynchia.ee.KeepingCollectorQueue(collectors=None, onclose=None)¶
