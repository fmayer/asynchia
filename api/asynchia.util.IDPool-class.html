Class IDPool

source code

object --+
         |
        IDPool

Pool that returns unique identifiers.

Identifierers obtained using the get method are guaranteed to not be returned by it again until they are released using the release method.

>>> pool = IDPool()
>>> pool.get()
0
>>> pool.get()
1
>>> pool.get()
2
>>> pool.release(1)
>>> pool.get()
1
>>>
