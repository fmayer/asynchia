Module maps

source code

Socket-maps are there to decide which sockets have I/O to be done and call the appropriate handle functions at the Handlers.

If select.poll is available, you will have PollSocketMap available and also assigned to DefaultSocketMap, which otherwise defaults to SelectSocketMap.

If select.epoll is available, you will have EPollSocketMap available and also assigned to DefaultSocketMap, which otherwise defaults to either PollSocketMap or SelectSocketMap.

If select.kqueue is available, you will have KQueueSocketMap available and also assigned to DefaultSocketMap, which otherwise defaults to one of the other three.
